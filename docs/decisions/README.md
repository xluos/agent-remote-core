# Architecture decision records

Each file in this directory is one ADR — an architectural decision that has
been made, the reasoning behind it, and the conditions under which it should
be revisited.

Format: lightweight ADR (numbered, status-flagged). Newer entries supersede
older ones only if they say so explicitly.
